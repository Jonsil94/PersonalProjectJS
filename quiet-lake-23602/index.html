    This website is part of an interactive installation that uses conversations to generate ecosystems. When you converse in this chat room, the statements you make create organisms in the installation. Each statement is analyzed for particular emotions using a lexicon created by the National Research Council of Canada. The sum of the emotions present in each statement is used to assign unique behaviours and appearances for the organisms created. Additional beings are created depending on the number of individuals currently on this website. For more information and process check out this link!
